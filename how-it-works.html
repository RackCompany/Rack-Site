How it works

We help users discover items across retailers and send them to the seller’s site to complete the purchase.

1) Product data ingestion

  • We use authorized sources (affiliate feeds, partner APIs, or licensed catalog providers).
  • We store product metadata such as title, brand, price, availability, category, and product URL.
  • We only display product images when usage is permitted by the relevant program terms.

2) Enrichment for better search

  • We normalize attributes (e.g., color families, materials, fits) for consistent filters.
  • We may use AI to generate structured tags from permitted text fields to improve search relevance.

3) Discovery + recommendations

  • Users browse a personalized feed based on saves, likes, and search activity.
  • Users can control personalization and reset recommendations.

4) Link-out purchase

  • When a user selects an item, we send them directly to the retailer’s product page.
  • We do not process payments or fulfill orders.
